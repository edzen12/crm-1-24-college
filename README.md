## CRM система для колледжа на Django Rest Framework'e
### после скачивания, выполните следующие команды в терминале
#### python -m venv venv
#### .\venv\Scripts\activate
#### pip install -r requirements.txt
#### python manage.py makemigrations
#### python manage.py migrate
#### python manage.py createsuperuser
#### python manage.py runserver
#### переход на api http://127.0.0.1:8000/api/
#### переход на админку http://127.0.0.1:8000/admin/
